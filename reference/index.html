Python 语言参考¶

版本:3.2
日期:五月 12, 2011

本参考手册描述了Python的语法和“核心语义”. 它很简洁, 但尽量的准确和完整. 内建函数, 模块和非必要的内建类型的语义在 Python 标准库 中描述. 对于一个非正式的语言介绍, 请参阅 Python 教程. 对于 C 或 C++ 程序员, 这有两个额外的手册: 扩展和嵌入 Python 解释器 描述了如何写一个 Python 扩展模块的高级概念, 而 Python/C API 参考手册详细描述了对 C/C++ 程序员可用的接口.
