关于¶

这些文档都是通过 Sphinx, 一个特别为 Python 文档所编写的文档处理器, 由 reStructuredText 源所生成.

该文档和它的工具链的开发在 docs@python.org 的邮寄列表里. 我们一直都在寻找志愿者来帮助维护文档, 给那发一封邮件时候轻松点!

特别感谢:

  • Fred L. Drake, Jr., 最初 Python 文档工具集的作者, 同时也是大量内容的作者;
  • 创造 reStructuredText 和 Docutils 套件的 Docutils;
  • Fredrik Lundh, 因为他的 Alternative Python Reference 项目, 在这个项目里, Sphinx 得到很多好点子.

参照 报告 Bugs 查阅如何报告该文档或 Python 自身的 bugs.
